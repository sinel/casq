Roadmap#

v0.1.0 (prototype)#

  • Skeleton

    • Local testing, linting & formatting

    • Documentation

    • Notebook support

    • Continuous integration

  • Qiskit foundation

  • Support for Qiskit dynamics

  • Pulse gates & circuit

  • Pulse backend & models

  • Pulse optimizer

  • Single-qubit gate optimization

  • Two-qubit gate optimization

v0.2.0 (functionality & verification)#

  • Noise model

  • Additional Hamiltonian models

  • Additional fidelity functions

  • Improved plotting & reporting

  • Verification

v0.3.0 (library support: qutip)#

  • Support for QuTiP (GRAPE, CRAB)

  • Improve pulse backend & model abstraction

v0.4.0 (library support: c3)#

  • Support for C3 (CMA-ES)

  • Improve pulse backend & model abstraction

v0.5.0 (ode solvers)#

  • Support for SciPy ODE solvers (jax?)

  • Evolutionary algorithms (leap, pygad, pymoo, pycma)

  • KROTOV (qucontrol/krotov)

  • GOAT (pmarkus-github/goat-quantumcontrol)

  • COCOA (code?)

  • Grid sweep for landscape

v0.6.0 (optimization)#
