src.casq.common.decorators.trace(*, log_entry: bool = True, log_exit: bool = True, level: str = 'TRACE') → Any#

Decorator for tracing entry to and exit from a function.

Args:

log_entry: Log entry to function if True. Default is True. log_exit: Log exit from function if True. Default is True. level: Log level. Default is TRACE.

Returns:

Wrapper function for decorator.

src.casq.common.decorators.timer(*, level: str = 'TRACE', unit: str = 'msec') → Any#

Decorator for timing the execution of a function.

Args:

level: Log level. Default is TRACE. unit: Time unit. Allowed values are {‘msec’, ‘sec’}. Default is ‘msec’.

Wrapper function for decorator.
