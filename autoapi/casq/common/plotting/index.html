casq.common.plotting.plot(configs: Sequence[LineConfig | LineCollectionConfig], figure: matplotlib.figure.Figure | None = None, hlines: list[tuple[float, str | None, LineStyle, matplotlib.axes.Axes | None]] | None = None, vlines: list[tuple[float, str | None, LineStyle, matplotlib.axes.Axes | None]] | None = None, title: str | None = None, legend_style: LegendStyle | None = None, show_grid: bool = False, filename: str | None = None, hidden: bool = False) → None#

Create and plot Matplotlib figure.

Args:

configs: Line configurations. figure: Optional Matplotlib figure to use for plotting. hlines: Horizontal line configurations. vlines: Vertical line configurations. title: Figure title. legend_style: Legend style. If None, then legend is hidden. show_grid: If True, use grid in plots. filename: If filename is provided as path str, then figure is saved as png. hidden: If False, then plot is not displayed. Useful if method is used for saving only.

Returns:

Matplotlib Figure.
